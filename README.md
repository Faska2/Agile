# Guide de Révision Technique

Ce projet est une page web interactive conçue pour faciliter la révision des concepts clés en développement logiciel moderne.

## Contenu

Le guide couvre les sujets suivants :
- **Méthodes Agile et Scrum** : Principes, rôles, événements et artefacts.
- **Jira** : Gestion de tickets, backlogs et suivi de projet.
- **Git et GitHub** : Commandes essentielles et flux de travail collaboratif.
- **DevOps** : Concepts CI/CD, pipelines et outils (Docker, Kubernetes, Jenkins).

## Structure du Projet

- `index.html` : La page principale contenant tout le contenu structuré.
- `style.css` : La feuille de style pour une mise en page propre, lisible et responsive.
- `text.txt` : Le contenu source original.

## Utilisation

Ouvrez simplement le fichier `index.html` dans votre navigateur web pour accéder au guide de révision.
